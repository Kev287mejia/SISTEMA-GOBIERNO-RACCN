# 🚀 Cómo Iniciar el Sistema (Modo Local)

Sigue estos pasos sencillos para levantar el sistema entorno de desarrollo.

## 1. Comandos Principales

Abre tu terminal en la carpeta del proyecto y ejecuta:

```bash
# 1. Instalar dependencias (solo si es la primera vez)
npm install

# 2. Sincronizar base de datos (recomendado si hubo cambios)
npx prisma generate

# 3. INICIAR EL SISTEMA
npm run dev

```

El sistema estará disponible en: **[http://localhost:3000](http://localhost:3000)**
